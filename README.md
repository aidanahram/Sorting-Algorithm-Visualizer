# Sorting Algorithm Visualizer

A Flutter project to examine what sorting algorithms are doing.

Can change the number of items in the list, speed, and algorithm

To run in debug mode:
  flutter run -d [device]

Algorithms included:
- [x] Selection Sort
- [x] Insertion Sort
- [x] Merge Sort
- [x] Bubble Sort
- [x] Quick Sort
- [x] Tim Sort
